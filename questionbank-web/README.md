# Question Bank Web

데스크톱 환경에서 문서를 업로드하고 텍스트를 추출해 문제은행을 구축하는 로컬 웹 도구입니다.

## 실행

```bash
python server.py
```

기본 주소는 `http://127.0.0.1:8765` 입니다.

## 지원 형식

- `txt`
- `docx`
- `pdf`

## 결과물 내보내기

상단의 `앱 가져오기용 JSON 내보내기` 버튼으로 Android 조회 앱에서 다시 불러올 문제은행 파일을 내려받을 수 있습니다.

## 배치 작업 흐름

1. 파일 또는 텍스트를 `배치 분석`합니다.
2. 한 파일 안의 여러 문제를 추출 결과 목록에서 한꺼번에 확인합니다.
3. `기존 문제은행 교체 저장` 또는 `기존 문제은행에 추가 저장`을 선택합니다.
4. `추출된 문제 전체 저장`으로 일괄 반영합니다.

즉, 한 문제씩 처리하는 방식이 아니라 한 파일에 있는 다수 문항을 한 번에 추출하고 저장하는 흐름입니다.

## 연동 방식

- 웹 DB와 앱 DB는 직접 연결하지 않습니다.
- 웹에서 문제은행을 관리한 뒤 JSON 번들을 수동으로 내보냅니다.
- Android 앱은 그 JSON 파일을 다시 불러와 내부 Room DB에 저장합니다.
- 즉, 구조는 `웹 SQLite -> JSON 파일 -> Android Room` 입니다.
