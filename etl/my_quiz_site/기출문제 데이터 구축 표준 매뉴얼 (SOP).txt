📑 기출문제 데이터 구축 표준 매뉴얼 (SOP)
전체 프로세스는 3단계로 구성됩니다. 각 단계가 성공해야 다음으로 넘어갈 수 있습니다.

Step 1. 데이터 클리닝 (기존 데이터 삭제)
새로운 회차의 기출문제를 넣기 전, 중복 방지를 위해 기존 데이터를 비웁니다.

터미널(CMD)에서 장고 프로젝트 폴더로 이동합니다.

python manage.py shell 실행.

아래 코드를 복사해서 붙여넣기:

Python
from quiz.models import Question
Question.objects.all().delete()
exit()
Step 2. AI 문서 분석 및 JSON 생성 (ultimate_parser.py)
비정형 문서(PDF, Word 등)를 AI가 읽어 정형 데이터(JSON)로 변환하는 핵심 단계입니다.

python ultimate_parser.py 실행.

파일 선택 창에서 분석할 파일(예: 2019년1회_기출.pdf)을 선택.

대기: 페이지당 약 20초의 휴식 시간이 포함되어 있으므로, 10페이지 기준 약 3~4분 소요됩니다.

확인: 터미널이 종료되면 폴더 내에 2019년1회_기출.json 파일이 생성되었는지 확인합니다.

주의: 파일 내용(JSON)을 열어 no, content, options가 규격에 맞게 들어갔는지 눈으로 살짝 확인하세요.

Step 3. 데이터베이스 적재 (db_uploader.py)
생성된 JSON 파일을 실제 서비스 DB(SQLite/PostgreSQL 등)에 밀어 넣는 단계입니다.

python db_uploader.py 실행.

방금 생성된 .json 파일을 선택.

"성공: OO개의 문제가 추가되었습니다"라는 알림창이 뜨면 완료.
