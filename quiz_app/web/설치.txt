1. 업로드 스크립트 작성 (db_uploader.py)
manage.py 파일이 있는 가장 바깥 폴더에 아래 파일을 새로 만드세요. (파일명: db_uploader.py)

Python
import os
import json
import django

# 1. Django 환경 설정 (내 프로젝트 이름을 가져옵니다)
os.environ.setdefault("DJANGO_SETTINGS_MODULE", "my_quiz_site.settings")
django.setup()

from quiz.models import Question, Choice

def insert_quiz_data(json_file_path):
    # 2. JSON 파일 읽기
    with open(json_file_path, 'r', encoding='utf-8') as f:
        data = json.load(f)

    print(f"{len(data)}개의 데이터를 DB에 넣기 시작합니다...")

    for item in data:
        # 3. Question(문제) 저장
        q = Question.objects.create(
            number=int(item['no']),
            content=item['content']
        )

        # 4. Choice(보기) 저장
        # 객관식 보기가 있는 경우에만 실행합니다.
        for opt in item.get('options', []):
            Choice.objects.create(
                question=q,
                choice_text=opt
            )
        
    print("모든 데이터가 성공적으로 저장되었습니다!")

if __name__ == "__main__":
    # 아까 생성된 JSON 파일 이름을 여기에 적으세요!
    target_json = "result_파일명.json" 
    insert_quiz_data(target_json)

Step 1: Poppler 다운로드
Poppler for Windows 다운로드 페이지에 접속합니다.

가장 최신 버전의 .zip 파일(예: poppler-24.xx.x.zip)을 다운로드합니다.

다운로드한 압축 파일을 원하는 곳에 풉니다.

추천 경로: C:\poppler (경로가 단순할수록 좋습니다.)

Step 2: bin 폴더 경로 복사
압축을 푼 폴더 안에 들어가면 Library -> bin 폴더가 있습니다.

그 bin 폴더까지 들어간 뒤, 상단 주소창을 클릭해 전체 경로를 복사합니다.

예: C:\poppler\Library\bin (이 안에 pdftocairo.exe 같은 파일들이 들어있어야 합니다.)

Step 2. AI 문서 분석 및 JSON 생성 (ultimate_parser.py)비정형 문서(PDF, Word 등)를 AI가 읽어 정형 데이터(JSON)로 변환하는 핵심 단계입니다.python ultimate_parser.py 실행.파일 선택 창에서 분석할 파일(예: 2019년1회_기출.pdf)을 선택.대기: 페이지당 약 20초의 휴식 시간이 포함되어 있으므로, 10페이지 기준 약 3~4분 소요됩니다.확인: 터미널이 종료되면 폴더 내에 2019년1회_기출.json 파일이 생성되었는지 확인합니다.주의: 파일 내용(JSON)을 열어 no, content, options가 규격에 맞게 들어갔는지 눈으로 살짝 확인하세요.Step 3. 데이터베이스 적재 (db_uploader.py)생성된 JSON 파일을 실제 서비스 DB(SQLite/PostgreSQL 등)에 밀어 넣는 단계입니다.python db_uploader.py 실행.방금 생성된 .json 파일을 선택."성공: OO개의 문제가 추가되었습니다"라는 알림창이 뜨면 완료.🌐 향후 개선: 웹 기반 통합 관리자 화면 (Admin UI)매번 터미널에서 파이썬 파일을 실행하는 게 번거롭다면, 장고(Django) 웹 화면에 '기출문제 업로드' 메뉴를 만들어 아래 기능을 통합할 수 있습니다.단계기능 구성 (Web UI)비고1. 파일 업로드브라우저에서 [파일 선택] 버튼으로 PDF/Word 업로드FileField 활용2. 실시간 분석웹 화면에 [AI 분석 시작] 버튼 배치백그라운드 작업(Celery 등)으로 실행3. 검수 및 수정AI가 분석한 내용을 표(Table) 형태로 미리 보여주고 오타 수정Editable Table 구현4. 최종 승인[DB 저장] 버튼을 누르면 최종 적재완료 메시지 출력💡 운영 팁모델 관리: 구글 API 정책에 따라 모델명이 바뀔 수 있으므로, 에러 발생 시 models.list()를 통해 현재 사용 가능한 최신 모델(gemini-2.5-flash-lite 등)로 수시 업데이트가 필요합니다.그림 데이터: AI가 그림을 텍스트로 잘 설명했는지 확인하려면 JSON의 content 필드를 체크하세요. 만약 설명이 부족하다면 ultimate_parser.py의 **프롬프트(Prompt)**를 "그림 내용을 더 자세히 설명해줘"라고 수정하면 됩니다.
